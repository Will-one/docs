## svg简介
svg 是一种基于XML的图像文件格式。Scalable Vector Graphics -- 可缩放矢量图

### 基本的SVG元素
先不深入看，初步了解下列使用的结构
* <svg> 包裹并定义整个矢量图。
* <line> 创建一条直线
* <polyline> 创建折线
* <rect> 创建矩形
* <circle> 创建圆
* <ellipse> 创建圆和椭圆
* <polygon> 创建多边形
* <path> 通过指定点以及点和点之间的线来创建任意形状

## svg基本使用
```html
<!-- 自闭合标签，默认宽高300*150 可以通过样式修改 -->
<svg></svg>
```
